# Event Management System

## Project Overview

Event Management System is a simple front-end web application developed using HTML, CSS, and JavaScript. It allows users to add, view, and manage event details through an interactive and user-friendly interface.

## Features

* Add new events
* Display upcoming events
* Event images
* Event date and location details
* Congratulations and announcement section
* Delete events
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* index.html
* style.css
* script.js

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
